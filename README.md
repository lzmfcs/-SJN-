# 冬雪SJN渲染器
作者：沈嘉南

## 介绍
SillyTavern 前端渲染控制插件。Saros社区来了越来越多的新人宝宝，
做了一个简洁的渲染器，
之后有问题请在dc找我。

## 重要声明
本插件**仅发布在 Saros 社区**
禁止二传、倒卖、商用
非官方渠道下载均为盗版，请举报

## 功能
- 渲染层数调节
- 帧率限制
- 消息保留数量
- 全局透明度
- 毛玻璃强度
- 圆角强度
- 动画/光晕/模糊开关
- 一键清空聊天
- 一键复制对话
- 一键重载界面

## 安装
1. 下载解压，将文件夹命名为 `dxxrq`
2. 放入：
SillyTavern/public/scripts/extensions/
3. 重启 SillyTavern
4. 在插件页面启用
